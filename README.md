# Spotify-Clone (진행 중)
## 목적
음악 플랫폼인 Spotify의 API를 활용하여 반응형 웹사이트 및 UI 개발.

또한 단일 페이지 어플리케이션(Single Page Application)을 구성하기 위해 react-router-dom 을 사용하여 구축

- **기간** :
07.08 ~ 

- **개발인원** :
1 명
